# 實驗紀錄

> 本文件是正式實驗索引。它只整理已知實驗與狀態，不重新執行驗證，也不做最終物理判斷。
>
> 重要警告：本文件中的「已完成」「已套用」多數來自 `docs/physics-engine-v2-plan.md` 原文記錄與 GLM 草稿整理，尚未逐一用目前 repo 重跑驗證。正式引用數字前，必須回查原文或重跑對應測試。

## 使用規則

- 實驗結果不得自動升格成模型決策。
- 已被後續取代的結果只能作歷史紀錄。
- 不確定日期、數字、測試範圍時，必須保留「不確定」。
- `return-studio.html` 與 `game4.html` 物理路徑不同，跨檔案引用實驗結果時必須標明檔案版本。
- `blend=0.9` 或高 blend 結果不可視為最終物理解。

## 實驗紀錄格式

正式補充單一實驗時，使用以下格式：

### Experiment ID
### 日期
### 目的
### 模型版本 / 涉及檔案
### 固定參數
### 搜尋參數
### 測試集
### 結果
### 失敗案例
### 狀態：已套用 / 已取代 / 研究中 / 待驗證 / 不確定
### 下一步

## 目前可視為高信任的實驗類型

高信任不代表不需重跑，只代表狀態比較清楚：

- 桌面接觸力學獨立 Node 驗證。
- 摩擦係數 `μ=0.13` 的工程校準紀錄。
- 發球旋轉換算成 rad/s 的方向。
- 回擊起點抬高 0.12m 的修正脈絡。

## 目前需最高警戒的實驗類型

- 任何與 push/chop 最佳參數相關的搜尋。
- 任何高 blend 或 `blend=0.9` 的結果。
- 任何來自 `return-studio.html` 研究功能、但尚未確認是否寫回 `game4.html` 的結果。
- 任何已標示被後續實驗取代的公式。
- 任何僅以「上桌/過網」為目標而未檢查旋轉方向的搜尋。
